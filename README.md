Ansible playbooks for publishing packages to NPM, PyPy and deployment to AWS, Heroku

## Setup

### Prerequisities

- python
- pip
- nvm
- npm
- node

````
pip install -r requirements.txt

npm install
````

### Usage

````
ansible-playbook <PLAYBOOK_NAME>
````
